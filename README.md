# Projet AlgoBox – Travail collaboratif

## Description du projet
Ce dépôt contient un projet d’algorithmique réalisé avec **AlgoBox** dans le cadre d’un travail collaboratif à deux.  
L’objectif est de concevoir, modifier et organiser plusieurs algorithmes tout en utilisant **GitHub** comme outil de travail en binôme.

## Membres du projet
- **Luca GAMASSA** : créateur du dépôt GitHub, gestion de la structure du projet et des fichiers initiaux  et finaux.
- **Antoine BOUCHEROY** : collaborateur principal, responsable de la gestion des modifications et de l’ajout de nouveaux fichiers.

## Organisation des fichiers
Les fichiers sont des algorithmes AlgoBox avec l’extension `.alg`.  
Chaque fichier porte un nom explicite indiquant clairement son rôle (exemple : calcul, tableau, boucle, pile ou face, etc.).

## Comment ouvrir et utiliser les fichiers
1. Installer **AlgoBox** sur votre ordinateur.
2. Cloner le dépôt GitHub ou télécharger les fichiers.
3. Les fichiers AlgoBox sont regroupés dans le dossier src/ afin de clarifier l’organisation du projet.
4. Ouvrir AlgoBox.
5. Cliquer sur **Fichier → Ouvrir**.
6. Sélectionner un fichier `.alg`.
7. Exécuter l’algorithme directement dans AlgoBox.

## Méthode de travail collaboratif
Le travail s’est fait selon le workflow suivant :
- Récupération des dernières modifications avec `git pull`.
- Modification ou ajout de fichiers `.alg` dans AlgoBox.
- Ajout des fichiers modifiés avec `git add`.
- Enregistrement des changements avec `git commit`.
- Envoi des modifications sur GitHub avec `git push`.

## Outils utilisés
- **AlgoBox** pour la création et l’exécution des algorithmes.
- **Git & GitHub** pour le suivi des versions et le travail collaboratif.

## Objectif pédagogique
Ce projet permet de comprendre :
- Le travail collaboratif avec GitHub.
- La gestion des versions d’un projet.
- L’organisation et la lisibilité d’algorithmes.
